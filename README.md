WhereIsTeacher
==============

An android application to help find the teacher.

App的名称为“老师在哪儿”，模仿时下流行的节目“爸爸去哪儿”，朗朗上口，便于记忆。
此App目的是为了方便学生查找老师办公室的位置。使用场景如下：
? 输入任意一位老师的姓名便可以搜索到老师办公室位置的详细信息以及老师的详细联系方式，并配以简洁直观的地图指引。
? 输入学校行政机关的名称（比如 计财处），便可以得到计财处所在的位置以及基本信息和相应的地图指引。
该应用极大地方便了同学与老师之间的沟通和交流，再也不用担心找不到老师在哪儿了，同时也免去了手工记忆各科老师和学校行政机关的联系方式的麻烦，应该会有比较好的使用前景。
